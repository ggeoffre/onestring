One String, Many Prompts: A Hands-On Workshop in AI Code Generation


One of the early ambitions of this initiative, before using artificial intellegence and large language models were even considerd, I wanted to solve the exact same problem over and over, with different languages, platforms, tools, and databases.  Utilizing Docker containers to streamline the setup process, the focus was always on the code itself.

Along the way, this became a vehicle to learn about artificial intellegence and large language models.  Not only did this accelerate the devlopment of this five to teh fith power of combinations, it also allowed me to explore the potential of AI in code generation.

As an example, looking at code snipets from five different implementations, one written in Circuitpython, one in Micropython, Arduino, Pico-SDK and ESP-IDF, we can also use AI/LLM to anayize the differences between teh implementations, platforms and langiuages.  Afterall that was the initial goal.
